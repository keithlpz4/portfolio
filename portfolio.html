Griffin's First Day of School

  • 2021
  • Book, Future Publishing House

Tools used

  • Pen
  • Pencil

Project description

Griffin's First Day of School is a children’s book published on April 1, 2021. The story is centered around a griffin learning the basics of being a creature of myth and legend. The author provided me with a script of each character’s mythical creature. I based my character design on characters from ancient drawings and descriptions of these mythical creatures. I submitted over 16 unique black and white illustrations, including the cover and back cover, which were then colored by the author.

You can more information on Griffin's First Day of School at the publisher's website: Future Publishing House.

Native-languages.org
Web Redesign


  • 2021
  • CMST 386 Principles of Web Design & Technology 2

Click here to view prototype click the picture below

Tools Used

  • HTML
  • CSS
  • Javascript
  • SASS
  • Draw.io

Project description

Native-languages.org, a small non-profit website that works to preserve and promote Native American Languages and cultures. Initially created in 1998, the site does not appear to contain CSS elements making it hard to read, navigate and access via mobile device. The goal of the project was to redesign the website adding new HTML5, CSS, and SEO elements to increase readability, user experience, and searchability.

This process included studying various encyclopedia sites like Encycolpedia.com, Britannica.com, and Wikipedia.com to understand popular layouts that display a massive amount of text information. Then created wireframes of the three common page types: An article page, a disclosure page, and a homepage, to build the prototype website.

USA Eagle Stamp

  • 2021
  • CMST 320 Illustration

Tools Used

  • Adobe Illustrator

Project description

The objective of this piece was to design a United States Postage stamp utilizing vector design software i.e., Adobe Illustrator. Corrugated edges and the letters the USA were requirements of the design. For this project, I used the color-blocking style to ensure all the parts of the stamp stood out. Additionally, I decided to make the theme of my piece centered around U.S. patriotism and chose the bald eagle and limit my use of colors.

County Park Moblie Application
UX Design

  • 2021
  • CMST 308 User Experience(UX) & Interface Design (ID)

To view protoype click here

Tools Used

  • Adobe XD

Project description

Design and build a prototype application using Adobe XD for the local state park to increase customer engagement, particularly with mobile users. To achieve this objective, I decided to recreate the Montgomery County parks website and condense it into a mobile application that allowed users to search the park for events, facilities, and park trails. I added a function that allowed park goers the ability to search and rent the park facilities. Up to 15 wireframes were created to demonstrate the design and functionality of the app.

Empty Pockets, Full Soul Magazine Spread

  • 2020
  • CMST 310 Fundamentals of Electronic Publishing

Tools Used

  • Adobe InDesign

Project description

The objective for this piece was to design a fictitious two-page magazine spread including a cover page however the requirement was that the articles must be “authentic writings” and NOT be randomly generated text. I decided to convert one of the travel blogs on the website Empty Pockets, Full Soul into the main article and utilize a few of their blog headers into covers lines. Image in the two-page spread was retrieved from the Empty Pockets, Full Soul website.


The main cover photo was retrieved from Sachin C Nair at pexels.com.

PJ SleepyTime Dance Party Poster

  • 2020
  • CMST 310 Fundamentals of Electronic Publishing

Tools Used

  • Adobe InDesign

Project description

Design a one-page dance poster for a fictional upcoming dance party. To create this poster, I used Person Wearing Bear Mask Dancing on Stage by Hygor Sakai on Pexels.com as it conveyed the energy of an over-the-top party to excite young would-be partiers. Vibrant font color was chosen to opposite dark image backgrounds with various font families for readability and to encourage the theme of enthusiasm.
